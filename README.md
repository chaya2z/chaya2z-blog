## What is this repo ?

This repo is source code of [my blog](https://chayanika.netlify.app/).

![homepage screenshot](/static/HomepageScreenshot.png)

Usage framework is React, static-site-generator is Gatsby.

## Environment

- Node.js 12.16.0
- React 16.13.1
- Gatsby 2.24.39

### Deploy

- Netlify
